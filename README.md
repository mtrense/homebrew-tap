# homebrew-tap

Homebrew tap for tools and projects built by [@mtrense](https://github.com/mtrense).

## Usage

```sh
brew tap mtrense/tap
brew install <formula>
```

Or install a formula directly without tapping first:

```sh
brew install mtrense/tap/<formula>
```

## Formulae

Formulae live in the [`Formula/`](Formula/) directory. They are published
automatically by release pipelines in the source repositories, which open
pull requests against this tap.
